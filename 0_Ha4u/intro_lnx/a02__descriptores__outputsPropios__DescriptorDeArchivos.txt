
	*** Descriptores de archivos reservados por el sistema ***

										0		stdin
										1		stdout
										2		stderr

					""  Libres del 3 - 9  ""

::: exec 3<> test_file

				<						'lectura'
				>   				'escritura'
				test_file   'archivo que va a contener todo el output redirigido'

		exec 3< test_file					"solo lectura File Descriptor 3"
		exec 3> test_file					"solo escritura File Descriptor 3"

		exec 3>> test_file				"solo escritura APPEND mode File Descriptor 3"
		exec 3<> test_file				"escritura/lectura y APPEND File Descriptor 3"

		exec 3>&-				"cerrar el descriptorDeArchivo/FileDescriptor fd"


		exec 4>&3-									"Crea una copia del fd 3 en el fd 4 y cierra el fd 3"
		exec 4>&3									"Crea una copia del fd 3 en el fd 4, los 2 apuntan al mismoArchivo"


### Listar los fd creador ...<3 <3 <3

	::: ls -l /proc/$$/fd							

					$$  "PID del shell actual...<3 <3 <3"


### Guardamos el output en el fileDescriptor fd
	::: whoami >&3					
									
							>& envia el stdout al descriptor de archivo
							3  Numero de descriptor de archivo


### Cat al fileDescriptor
	::: cat <&3
